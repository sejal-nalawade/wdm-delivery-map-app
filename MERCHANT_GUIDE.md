# Delivery Locations Map - Merchant Guide

Welcome! This guide will help you set up and customize your delivery locations map.

## Quick Start (5 Minutes)

### Step 1: Install the App
1. Click "Add app" from the Shopify App Store
2. Review and approve the requested permissions
3. Click "Install app"

### Step 2: Configure Your Delivery Zones
1. Open the app from your Shopify admin
2. You'll see three tabs: **Map Configuration**, **Button Customization**, and **Preview**
3. Start with **Map Configuration**:
   - **Same Day Delivery Map**: Choose "Default NYC Map" or upload your own
   - **Scheduled Delivery Map**: Choose "Default USA Map" or upload your own
   - Edit the description text for each delivery type
4. Click **Save Settings**

### Step 3: Customize Button Appearance
1. Click the **Button Customization** tab
2. Customize:
   - Button text (e.g., "Same Day" vs "Next Day")
   - Active/inactive button colors
   - Button alignment (left, center, or right)
   - Button shape (square, rounded, or pill)
   - Default active mode (which map shows first)
3. Click **Save Settings**

### Step 4: Add to Your Theme
1. Go to **Online Store** â†’ **Themes** in your Shopify admin
2. Click **Customize** on your active theme
3. Navigate to the page where you want the map (usually Homepage)
4. Click **Add section**
5. Select **"Delivery Locations Map"** from the Apps section
6. Optionally edit the section title
7. Click **Save**

That's it! Your delivery map is now live on your store.

## Detailed Configuration

#### Same Day Delivery Map
- **Default NYC Map**: Pre-designed map highlighting the NYC metropolitan area

#### Scheduled Delivery Map

- **Custom Image**: Upload your own map image

### Uploading Custom Maps

#### Image Requirements
- **Format**: PNG, JPG, or SVG

## Troubleshooting

### Map Not Showing on Storefront
**Check:**
1. Is the app block added to your theme?
2. Are settings saved in the admin?
3. Is the theme published (not in preview)?
4. Try clearing your browser cache

### Custom Image Not Loading
**Check:**
1. Is the image URL correct?
2. Is the image publicly accessible?
3. Is the file size under 2MB?
4. Try re-uploading the image

### Buttons Not Switching Maps
**Check:**
1. Are both maps configured?
2. Try saving settings again
3. Clear browser cache and reload
4. Check browser console for errors

### Colors Not Applying
**Check:**
1. Did you click "Save Settings"?
2. Try using hex color codes (e.g., #FF0000)
3. Clear browser cache
4. Refresh the theme editor

## FAQ

### Can I have more than two delivery options?
Currently, the app supports two modes: Same Day and Scheduled. For more complex needs, contact support.

### Can I use the same map for both modes?
Yes! Just upload the same image or select the same default map for both.

### Will this affect my checkout shipping rates?
No, this is a visual display only. Configure actual shipping rates in Shopify Settings â†’ Shipping.

### Can I customize the map colors?
For default maps, colors are preset. Upload custom images for full color control.

### Does this work with all themes?
Yes! The app uses Online Store 2.0 app blocks, compatible with all modern Shopify themes.

### Can I add this to multiple pages?
Yes! Add the block to any page that supports sections in your theme editor.

### Is it mobile responsive?
Yes! The map automatically adjusts for mobile, tablet, and desktop screens.

### Can I hide the description text?
Yes! Uncheck "Show description text" in Map Configuration.
